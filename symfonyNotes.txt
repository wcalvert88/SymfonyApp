symfony notes

debug container
	php bin/console debug:container 'container to debug here'
	brings up the debug:container for container specified

composer server OR symfony/web-server-bundle
	./bin/console server:run

logger
	tail -f var/log/dev.log

mysql://db_user:db_password@127.0.0.1:3306/db_name

doctrine
	php bin/console doctrine:migrations:diff
	this builds a version file with commands to build and drop table

	php bin/console doctrine:migrations:migrate
	this makes changes to the table from your version file

	php bin/console doctrine:migrations:migrate prev
	this reverts to previous version

	php bin/console doctrine:migrations:migrate next
	this brings back the next version

	both can end with --no-interaction to get rid of asking are you sure

	composer require --dev doctrine/doctrine-fixtures-bundle
	can create fixtures which are fake data
